Based on Tobin James, The World Wide Web (WWW), known as the Web, is an information system where documents and other web resources are identified by Uniform Resource Locators (URLs), which may be interlinked by hyperlinks, and are accessible over the Internet.

In this page we will explore concepts related to Web's function.

 

The Internet

The Internet is practically the tool that connects computers (university, governmental etc) all around the world, creating eventually a network.

So how does the Internet work?

Apart from the connection, the Internet makes sure to choose the way messages from one computer to another will be passed on, creating protocols, which are eventually being broken down to pieces called packets.

Protocols? Packets? What are these? Before we jump on this terms, let's watch this video on how the internet works.

 

Internet Protocols

As we saw, internet protocols are rules that determine how messages will be addressed and passed on in the network of computers.

Packets (TCP/UDP)

However, if the message is too big, protocols are broken down in packets. Each packet has information about how packets fit together, as well as information about the sender and the recipient of the message.

TCP and UDP are the most common protocols. The following video will give a general overview about their function.


IP Addresses

Each computer on the internet (yes, that is right, even the own on which you are browsing this webpage) has a unique numeric address called IP address.

Are you curious about your IP? Find it in this Domain server, which is something like an address book.

Routers

Routers are the guides. In other words, their purpose is to "guide" the packets through the internet. As soon as the packets arrive to their destination, the information embedded in each packet tell how to put all the packets back together.


DNS

DNS is short for Domain Name System and it is a translation system we cam use to search on the internet. In other words, when we search for Facebook or Wikipedia or any domain name, typing in human language, the browser checks is one of them has the domain's IP address.

HTTP

HTTP or Hyper Text Transfer Protocol is an application which defines the message structure between the browser and the webserver that is actually deliver this application. Thanks to WWW in 90s, HTML was develop (basically the process of creating a link on a text) that led to HTTP.

HTML

Last but not least, HTML (Hyper Text Markup Language) is the actual content we see in a website. HTML makes it for us to click on links on a webpage.
